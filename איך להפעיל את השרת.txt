כדי להפעיל את השרת אצלכם :
1.תורידו את הקובץ holontelecomdb_users מהגיט מחדש, הוא יהיה בתיקייה של הפרויקט.
2. בmysqlworkbench תלחצו על file>new model > choose holontelecomdb
3. למעלה יופיעו לכם server, תלחצו עליו ותבחרו data import
4.תבחרו ב import from self contained file ותבחרו בholontelecomdb_users
5.תלחצו על start import
אחרי זה כל הפרטים של השרת יהיו אצלכם ולא צריך לעדכן משהו מעבר
תוודאו שאתם יכולים להריץ שאילתות ורואים את הנתונים שכבר נשמרו אצלי
